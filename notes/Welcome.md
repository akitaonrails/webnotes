# Welcome to WebNotes

A simple, self-hosted markdown note-taking app.

## Features

- **File-based storage** - Notes are stored as markdown files
- **Nested folders** - Organize notes in hierarchies
- **Live preview** - Toggle with `Ctrl+Shift+P`
- **Dark/Light mode** - Click the theme toggle in the header
- **Auto-save** - Changes are saved automatically
- **Keyboard shortcuts**:
  - `Ctrl+N` - New note
  - `Ctrl+S` - Save now
  - `Ctrl+Shift+P` - Toggle preview

## Getting Started

1. Click the **+** button in the sidebar to create a new note
2. Click the folder icon to create a new folder
3. Right-click on items for rename/delete options
4. Start writing in markdown!
